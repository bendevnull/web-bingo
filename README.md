# Multiplayer Bingo

This is a complete multiplayer Bingo game that allows players to join a game and play in real-time.

## Features
- Join a game lobby
- Real-time gameplay with multiple players
- Display of current numbers and called numbers
- Winner announcement

## Setup
To run this project, clone the repository and open `index.html` in a web browser. You'll also need a server to host the socket.io connection.
